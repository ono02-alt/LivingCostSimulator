# 🏙 生活費・住環境シミュレーター

水道・電気・賃貸相場を全国比較できるWebアプリです。  
人口10万人以上の全国140都市以上に対応。**完全無料・ビルド不要**で動作します。

🔗 **公開URL（設定後に更新）**  
`https://ono02-alt.github.io/LivingCostSimulator/`

---

## 主な機能

| 機能 | 内容 |
|------|------|
| 💧 水道シミュレーション | 使用量スライダーで上下水道料金をリアルタイム試算 |
| ⚡ 電気料金比較 | 地域の電力会社プランを安い順3位表示・詳細シミュレーション |
| 🏠 賃貸相場 | 間取り別相場＋参考物件例3件（安い順） |
| 📊 地域情報 | 利便性・治安・物価・気候・年収・生活保護承認率 |
| 🤖 AI概要 | Gemini APIで地域の生活環境を自動解説（任意） |

---

## 完全無料の仕組み

- **GitHub Pages**：静的ファイルのホスティングが無料
- **ビルド不要**：`index.html` 1ファイルのみ。Node.jsやnpm不要
- **Gemini API（任意）**：無料枠1,500リクエスト/日（クレカ不要）
  - アプリ内で1日30回に制限し、超過時は自動停止
  - 有料課金は一切発生しない設計
- **AI未設定でも全機能使用可能**（水道・電気・賃貸・地域情報はすべてローカル動作）

## 📱 スマホだけで公開する手順

### ステップ1：GitHubアカウント作成（既にある場合はスキップ）

1. ブラウザで [github.com](https://github.com) を開く
2. 「Sign up」をタップ
3. メールアドレス・パスワード・ユーザー名を入力して登録

---

### ステップ2：リポジトリを作成する

1. GitHubにログインし、右上の **「＋」** ボタンをタップ
2. **「New repository」** を選択
3. 以下のように入力する

   | 項目 | 入力内容 |
   |------|----------|
   | Repository name | `living-cost-simulator` |
   | Public / Private | **Public**（必須） |
   | Add a README | チェックしない |

4. **「Create repository」** をタップ

---

### ステップ3：ファイルをアップロードする

1. 作成したリポジトリのページを開く
2. **「uploading an existing file」** または **「Add file → Upload files」** をタップ
3. `index.html` と `README.md` の2ファイルを選択してアップロード
4. 下部の **「Commit changes」** をタップ

---

### ステップ4：GitHub Pagesを有効にする

1. リポジトリページの **「Settings」**（設定）をタップ
2. 左メニューから **「Pages」** を選択
3. **Source** の欄で以下を選択

   - Branch：**main**
   - フォルダ：**/ (root)**

4. **「Save」** をタップ
5. 数分後に以下のURLでアクセスできる

   ```
   https://あなたのユーザー名.github.io/living-cost-simulator/
   ```

---

### ステップ5：Gemini APIキーを取得する（任意・AI機能を使う場合のみ）

1. ブラウザで [aistudio.google.com](https://aistudio.google.com) を開く
2. Googleアカウントでログイン
3. 左メニューの **「Get API key」** をタップ
4. **「Create API key」** をタップしてキーをコピー
5. 公開したアプリを開き、右上の **「⚙️ APIキー設定」** からキーを貼り付けて保存

> **注意：** APIキーはあなたのブラウザ（localStorage）にのみ保存されます。  
> GitHubに公開されることはありません。他人に教えないようにしてください。

---

### ステップ6：ファイルを更新したいとき

1. GitHubのリポジトリページを開く
2. `index.html` をタップ
3. 右上の **鉛筆アイコン（Edit）** をタップ
4. 内容を編集して **「Commit changes」** をタップ
5. 数分後に反映される

## データについて

- 水道・電気料金、賃貸相場は **公開統計・調査データに基づく概算値**（2024〜2025年時点）
- 電気プランは **キャッシュバック・割引を除く通常料金体系のみ** で比較
  （キャッシュバック情報は参考として別途表示）
- 物件情報は相場ベースの参考データです。実際の物件は SUUMO・HOME'S 等でご確認ください
- 地域情報（治安・年収・承認率等）は公開統計に基づく参考値です

---

## 注意事項

- 本ツールは情報提供を目的とした参考ツールです
- 正確な料金・情報は各自治体・電力会社・不動産会社にてご確認ください
- Gemini API の無料枠の条件は Google の方針変更により変わる場合があります
